# CLI Docker image for Drupal

Based on Debian 7.0 "Wheezy" (debian:wheezy)

## Includes

- php-fpm and php-cli 5.6.x
- composer 1.0-dev
- drush 6,7,8
  - registry_rebuild
  - coder-8.x + phpcs
- drupal console 0.9.7
- ruby 1.9.3 + gem 1.8.11
- bundler 1.8.5
- nodejs 4.2.2 + npm 3.4.0
- bower 1.6.5
- grunt 0.1.13
- gulp 3.9.0
- python 2.7.3

Other notable tools:

- git
- curl/wget
- zip/unzip
- mysql-client
- imagemagick
- ping
- mc
